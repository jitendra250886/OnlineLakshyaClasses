Basic Computer Skills Course

Course Overview

This course is designed to introduce students to fundamental computer operations, providing a solid foundation for digital literacy and effective computer use.

Course Modules

1. Introduction to Computers

An overview of different types of computers, their components, and their roles in our daily lives.

2. Using a Keyboard and Mouse

Hands-on exercises to familiarize students with keyboard shortcuts, mouse operations, and basic typing skills.

3. Basic Software Applications

Introduction to commonly used software applications, including word processors, spreadsheets, and presentation tools.

4. Digital Literacy

Understanding online etiquette, internet safety, and responsible technology use.

Daily Required Work Software

  • Web Browsers (Chrome, Firefox, Safari)
  • Email Clients (Outlook, Gmail)
  • Word Processing Software (Microsoft Word, Google Docs)
  • Spreadsheet Software (Microsoft Excel, Google Sheets)
  • Presentation Software (Microsoft PowerPoint, Google Slides)
  • Calendar and Scheduling Tools

Top Software Companies and Products

  3. Google - Chrome OS, G Suite
